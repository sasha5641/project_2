# Фінансовий Симулятор (Financial Simulator)

Цей проєкт - це консольний фінансовий симулятор, написаний мовою C++. Програма дозволяє керувати віртуальними фінансами, контролювати доходи та витрати, користуватися кредитною карткою та навіть заробляти гроші на "фрілансі". Програма має систему збереження прогресу та історії дій на картці.

## Що є в програмі:
* **Два типи рахунків:** Зарплатна картка для власних коштів та кредитна картка із заданим лімітом.
* **Реалістична система кредитування:** Можна брати гроші в борг, погашати його частинами. На залишок боргу щомісяця автоматично нараховуються відсотки (2%). Стан кредитки відображається зручним візуальним прогрес-баром.
* **Категоризація витрат:** Усі витрати записуються за категоріями (Їжа, Транспорт, Одяг, Техніка, Розваги тощо).
* **Симуляція часу:** Можливість перемотувати дні. При переході на новий місяць гра автоматично нараховує зарплату, знімає фіксовану суму за комунальні послуги та розраховує відсотки по кредиту.
* **Мінігра "Фріланс":** Інтегрована гра "Шибениця". Якщо відгадати слово, гравець отримує грошовий бонус на свою зарплатну картку.
* **Збереження та звіти:** Весь прогрес, баланси та історія транзакцій автоматично зберігаються у файл `bank_data.txt` для підвантаження даних. Також програма генерує текстовий файл `financial_report.txt`, який виглядає як виписка з банку.
* **Система авторизації:** При першому запуску створюється профіль з паролем, який захищає вхід до системи.

---

## Що планується додати в майбутньому:
* **Депозити:** Можливість покласти зароблені гроші на депозитний рахунок під відсоток, щоб отримувати пасивний дохід.
* **Випадкові події:** Раптові витрати або доходи під час перемотування часу (наприклад, "Зламалася машина — витрата 5000 грн" або "Видали премію — дохід 2000 грн").
* **Мініігри:** Додати інші способи заробітку на фрілансі, крім "Шибениці".

---

## Як запустити:
1. Завантажте всі файли з кодом (`.cpp` та `.h`).
2. Відкрийте їх у будь-якому середовищі для C++.
3. Скомпілюйте та запустіть програму.

**Нюанс:** у проєкті використовується бібліотека `windows.h` для зміни кольорів тексту в консолі та правильного відображення українських літер, тому запускати програму потрібно на ОС Windows.

---

## Як користуватися:
* **Перший запуск:** Програма попросить придумати пароль (без пробілів), ввести ваш початковий баланс, розмір щомісячної зарплати та суму комуналки.
* **Навігація:** В головному меню просто вводьте цифру відповідного пункту та натискайте `Enter`.
* **Транзакції:** При витратах чи взятті кредиту вводьте суму в гривнях (програма сама розраховує копійки). Всі операції захищені — ви не зможете витратити більше, ніж є на рахунку, або перевищити кредитний ліміт.

---

## Які бібліотеки використовував:
* `<iostream>` та `<string>` — для виводу тексту та роботи з рядками.
* `<fstream>` — для збереження даних профілю у текстові файли та генерації звітів.
* `<iomanip>` — для красивого форматування виводу (вирівнювання тексту, коректне відображення копійок з нулями, наприклад `05` копійок).
* `<cmath>` — для математичних операцій (наприклад, знаходження модуля числа при роботі з від'ємними балансами).
* `<windows.h>` — для налаштування кодування консолі, встановлення кольорів для різних типів операцій (зелений для доходів, червоний для витрат) та створення пауз (`Sleep`).
